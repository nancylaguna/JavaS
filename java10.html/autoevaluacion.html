    10.1 Complete las siguientes oraciones:

  1. Es posible almacenar listas y tablas de valores en arreglos
  2. El número que se utiliza para referirse a un elemento específico de un arreglo se conoce como su índice
  3. El proceso de poner en orden los elementos de un arreglo se conoce como ordenamiento el arreglo.
  4. Al proceso de determinar si un arreglo contiene cierto valor clave se le conoce como búsqueda el arreglo.
  5. Un arreglo que utiliza dos índices se conoce como arreglo bidimensional

    10.2 Indique si cada uno de los siguientes enunciados es verdadero o falso. Si es falso, explique por qué.

  1. Un arreglo puede almacenar muchos tipos diferentes de valores.
    Verdadero
  2. Por lo general el índice de un arreglo debe ser un valor de punto flotante.
    Falso. El índice de un arreglo debe ser un entero o una expresión entera.
  3. Un elemento individual de un arreglo que pasa a una función y se modifica en ésta debe contener el valor modificado cuando la función invocada complete su ejecución.
    Falso. Los elementos individuales de un tipo de datos primitivo se pasan por valor. Si se pasa una referencia a un arreglo, entonces las modificaciones a los elementos del arreglo se reflejan en el elemento original del arreglo. Además, el elemento individual de un tipo de objeto que se pasa a una función se pasa por referencia, y los cambios en el objeto se reflejarán en el elemento del arreglo original.

    10.5 Encuentre el o los errores en cada uno de los siguientes segmentos de programa y corríjalos.

  1. var b = new Array(10);
