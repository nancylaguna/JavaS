12.1 Indique si cada uno de los siguientes enunciados es verdadero o falso. Si es falso, explique por qué.

  1. Cada elemento de HTML5 en una página se representa mediante un árbol DOM. Falso. Todo elemento se representa mediante un nodo DOM. Cada nodo es miembro del árbol DOM del documento
  2. Un nodo de texto no puede tener nodos hijos. Verdadero
  3. El nodo document en un árbol DOM no puede tener nodos hijos. Falso. El documento es el nodo raíz, por lo tanto no tiene nodo padre.
  4. Es posible cambiar la clase de estilo de un elemento en forma dinámica, estableciendo el atributo style Falso. La clase de estilo se modifica estableciendo el atributo class.
  5. El método createElement crea un nuevo nodo y lo inserta en el documento. Falso. El método createElement crea un nodo pero no lo inserta en el árbol DOM
  6. El método setlnterval llama a una función repetidas veces, con un intervalo de tiempo establecido. Verdadero
  7. El método InsertBefore se llama en el objeto documento; recibe un nuevo nodo y uno existente para insertar el nuevo antes. Falso. insertBefore se llama en el padre
  8. El intervalo más reciente iniciado se detiene al momento de llamar al método clearlnterval. Falso. clearlnterval recibe un identificador de inetrvalo como argumento para determinar qué intervalo terminal.
  9. La colección links contiene todos los vínculos en un documento con un atributo id especificado. Falso. La colección links contiene todos los vínculos en un documento.

    12.2 Complete las siguientes oraciones

  1. La propiedad innerHTML se refiere al texto dentro de un elemento, incluyendo etiquetas de HTML5.
  2. El árbol DOM de un documento representa todos los nodos en un documento, así como las relaciones entre ellos.
  3. La propiedad lenght contiene el número de elementos en una colección.
  4. El método item permite el acceso a un elemento individual en una colección.
